## Installation

```sh
$ npm install -g express-restapi-es6
```

## Quick Start

Create the app:

```bash
$ express-restapi-es6 yournameapp
```

Install dependencies:

```bash
$ cd yournameapp
$ npm install
```

Start your app at `http://localhost:3000/api/value`

```bash
$ npm start
```
Development mode (Nodemon)
```bash
$ npm run dev
```
Run test
```bash
$ npm run test
```
